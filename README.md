# calculadora-imc
Uma calculadora de IMC simples

Este é um projeto simples de calculadora de IMC criado para testes, utilizando **HTML**, **CSS** e **JavaScript**.

## Sobre o projeto
Esta calculadora é um projeto de teste que usa informações de altura e peso dadas pelo usuário para trazer como resultado o valor do IMC calculado e a classificação (de 'abaixo do peso' até 'obesidade III')

## Objetivo
O objetivo do projeto é a prática de conhecimentos adquiridos em HTML, CSS e JS, mas é funcional e pode ser utilizada com o propósito de descobrir sua classificação de IMC.
